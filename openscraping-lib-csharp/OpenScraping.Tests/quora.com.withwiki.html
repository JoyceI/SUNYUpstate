    3. Customize your taskbar in Windows 7 & 8:

    4. All useful commands in RightClick: Download - FileMenu Tools.


    5. Windows logon background:

    In Windows Run, go to %WINDIR%\System32\oobe\info\backgrounds\,

    6. Create a shortcut for Windows Shutdown & abort:

    Right click, create a new shortcut, type shutdown.exe -s -t 3600 and save it. Click on the shortcut and Windows will shutdown after 1 hour (the number is in seconds). Now go to Rainy Mood. Windows will shutdown after you fall asleep enjoying the rain. Create another shortcut with command shutdown.exe -a to abort shutdown caused by accidental click of the previous shortcut.



    7. Open notepad in web browser:

    Type data:text/html, <html contenteditable> in the address bar of a web browser, and you have a temporary notepad .. like a clipboard.
